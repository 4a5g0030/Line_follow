# line_follow
2019 Taiwan TDK Robocon<br>
使用滑行視窗把橫向中最有可能為線的地方框出來，並且以該行的中心點為基準可以得出相對位置。</br>
但因為滑行視窗的 bigO 為 N^2 執行速度非常慢。<br>
40s 的影片平均要跑 130s 左右<br>
